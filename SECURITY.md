# Security Policy

## Supported Versions

이 프로젝트에서 배포하는 버전 중 보안 취약점 검사를 수행한 버전은 다음과 같습니다.

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |


## Reporting a Vulnerability

만약 새로운 보안 취약점을 발견하신 경우에는 [이슈 등록하기](https://github.com/hamonikr/hamonize/issues)를 통해 알려주세요.
