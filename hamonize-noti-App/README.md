# Hamonize-Noti

Hamonize-Noti는 Application Alert 프로그램입니다.

![image](https://github.com/hamonikr/hamonize/blob/master/img/hamonize-notiImg.png)
<br></br>
###  Usage
```
hamonize-noti.AppImage <message>
```

### Install 
```
git clone
cd ./hamonize-noti
npm start
```


### Build
```
npm run build:linux64

```
